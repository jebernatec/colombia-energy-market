### Introduction

Colombia Energy Market Analysis project using the XM API

### Clone the project
1. clone the repository
``` Bash
git clone https://github.com/jebernatec/energy_market.git
```
2. Install dependencies
``` Bash
pip install -r requirements.txt
```
3. Run jupyter
``` Bash
jupyter notebook
```
